# Jjodel Documentation

This repository contains the official documentation for 
[Jjodel](https://github.com/jjodel-modeling/jjodel) — 
an open-source web-based metamodeling tool for research and education.

## Structure

- `panels/` — contextual help for in-app panels
- `tutorials/` — step-by-step guides
- `reference/` — formal language specifications (JjEL, JjTL, JjScript)
- `concepts/` — foundational MDE concepts
- `examples/` — annotated example metamodels

## How this is used

- **In-app help** — Jjodel's HelpDrawer fetches these files at runtime
- **Documentation site** — rendered via VitePress at docs.jjodel.io (coming soon)
- **Jjodie AI assistant** — uses these files as knowledge base

## Contributing

Documentation contributions are welcome. You don't need to know the 
Jjodel codebase — just open a PR on the relevant Markdown file.
