# Dashboard

The Dashboard is your personal hub in Jjodel. It is the first screen you see after logging in and provides an overview of all your projects, recent activity, and quick access to key actions.

## Overview

<!-- TODO: screenshot — dashboard overview (new UI) -->

The Dashboard displays your projects as cards or a list view. Each project card shows the project name, description, and last modification date. From here you can:

- **Open** an existing project by clicking on its card
- **Create** a new project via the main menu
- **Delete** or **rename** projects
- **Search** through your projects

## Main Menu

The main menu is accessible from the upper-left corner of the Dashboard. It provides access to:

- **New Project** — create a new empty project
- **New Metamodel** — create a standalone metamodel
- **New Model** — create a model within an existing project

## Project Cards

Each project card represents a self-contained workspace. A project in Jjodel contains:

- One or more **metamodels** defining the language structure
- One or more **models** conforming to those metamodels
- One or more **viewpoints** defining visual representations

Click on a project card to enter the project workspace and start editing.

## Recent Activity

The Dashboard highlights your most recently modified projects, making it easy to resume work where you left off.

:::note
All projects are stored in the cloud. You can access them from any device by logging into your account.
:::
