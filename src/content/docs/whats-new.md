## v0.1.0 -- April 2025

**New**
- Cloud-native workspace with browser-based metamodel editor, model editor, and viewpoint designer.
- Sign-in, registration, and password recovery flows with email verification.
- Project system for organizing metamodels, models, and viewpoints in a single workspace.
- Metamodel editor with support for classes, attributes, enumerations, and references.
- Viewpoints for defining concrete syntax and visual representations of metamodel elements.
- JjEL (Jjodel Expression Language) for writing constraints and derived features.
- JjTL (Jjodel Transformation Language) for model-to-model transformations.
- Dashboard with project listing, recent activity, and quick-create actions.
- Console panel for runtime feedback, validation messages, and expression evaluation.
- Tree views for navigating model and metamodel hierarchies.

**Improved**
- Streamlined getting-started flow: from account creation to first working project in minutes.

**Fixed**
- Initial public release -- no prior issues to report.
